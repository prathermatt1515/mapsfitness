# MAPsFitness

Personal fitness and nutrition tracker. Mobile-first PWA, built and used by Matt Prather.

> Discipline to Be Better. **#RELENTLESS**

---

## Stack

- **Hosting:** GitHub Pages (static)
- **Auth & data:** Firebase (Auth + Firestore, free tier)
- **Food database:** USDA FoodData Central (proxied via Cloudflare Worker)
- **Frontend:** Vanilla HTML / CSS / JS, no build tooling
- **Installable:** PWA with iOS home screen support

## Live

https://prathermatt1515.github.io/mapsfitness

## Build phases

- **Phase 0:** Foundation — auth, brand shell, PWA install, empty data structure
- **Phase 1:** Workouts — lift logging, cardio logging, weekly + today views, seeded program
- **Phase 2:** Nutrition — USDA food search, macro tracking, recipes, water
- **Phase 3:** Progress — weight, VO2 max, charts, exports
- **Phase 4:** Coaching — progression suggestions, calorie burn integration
